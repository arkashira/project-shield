# Partner Targets
## Introduction
Project Shield aims to integrate with key SaaS and API platforms to enhance its functionality and provide a seamless experience for open source project maintainers and contributors. The following partner targets are prioritized based on their potential to add value, ease of integration, and revenue-sharing opportunities.

## Partner Integration Roadmap
The integration roadmap is divided into three phases, with Phase 1 focusing on high-priority partners with affiliate or revenue-share opportunities.

### Phase 1 (High Priority)
1. **GitHub**: Integrate Project Shield with GitHub to enable seamless project scope management and automated issue tracking.
	* Free-tier limits: 1000 API requests per hour
	* Integration effort: M (Medium)
	* Value-add: Simplifies project scope management and issue tracking for GitHub users
	* Affiliate/revenue-share: Yes, through GitHub Marketplace
2. **GitLab**: Integrate Project Shield with GitLab to provide project scope management and automated issue tracking for GitLab users.
	* Free-tier limits: 500 API requests per minute
	* Integration effort: M (Medium)
	* Value-add: Enhances project scope management and issue tracking for GitLab users
	* Affiliate/revenue-share: Yes, through GitLab Integrations
3. **Trello**: Integrate Project Shield with Trello to enable visual project scope management and automated board updates.
	* Free-tier limits: 100 API requests per minute
	* Integration effort: S (Small)
	* Value-add: Provides a visual representation of project scope and automates board updates for Trello users
	* Affiliate/revenue-share: Yes, through Trello Power-Ups
4. **Asana**: Integrate Project Shield with Asana to enable automated task management and project scope tracking.
	* Free-tier limits: 1000 API requests per minute
	* Integration effort: M (Medium)
	* Value-add: Streamlines task management and project scope tracking for Asana users
	* Affiliate/revenue-share: Yes, through Asana Integrations
5. **Notion**: Integrate Project Shield with Notion to provide a centralized project scope management and note-taking solution.
	* Free-tier limits: 100 API requests per minute
	* Integration effort: S (Small)
	* Value-add: Enhances project scope management and note-taking capabilities for Notion users
	* Affiliate/revenue-share: Yes, through Notion Templates

### Phase 2 (Medium Priority)
1. **Jira**: Integrate Project Shield with Jira to enable automated issue tracking and project scope management for Jira users.
	* Free-tier limits: 1000 API requests per hour
	* Integration effort: L (Large)
	* Value-add: Simplifies project scope management and issue tracking for Jira users
	* Affiliate/revenue-share: No
2. **Microsoft Teams**: Integrate Project Shield with Microsoft Teams to provide a centralized project scope management and communication solution.
	* Free-tier limits: 1000 API requests per minute
	* Integration effort: M (Medium)
	* Value-add: Enhances project scope management and communication capabilities for Microsoft Teams users
	* Affiliate/revenue-share: No

### Phase 3 (Low Priority)
1. **Basecamp**: Integrate Project Shield with Basecamp to enable automated project scope management and team collaboration.
	* Free-tier limits: 100 API requests per minute
	* Integration effort: S (Small)
	* Value-add: Simplifies project scope management and team collaboration for Basecamp users
	* Affiliate/revenue-share: No
2. **Wrike**: Integrate Project Shield with Wrike to provide automated project scope management and task tracking.
	* Free-tier limits: 1000 API requests per minute
	* Integration effort: M (Medium)
	* Value-add: Enhances project scope management and task tracking capabilities for Wrike users
	* Affiliate/revenue-share: No

By prioritizing integrations with affiliate or revenue-share opportunities, Project Shield can generate additional revenue streams and enhance its value proposition for users.
