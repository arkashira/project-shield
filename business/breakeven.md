# breakeven.md

## Cost per Active User
- Compute: $3.00/user/month (server, database, AI processing)
- Storage: $1.00/user/month (project documentation, scope definitions)
- Bandwidth: $1.00/user/month (application delivery, file transfers)
- **Total Cost per Active User: $5.00/user/month**

## Pricing Tiers
| Tier | Price/Month | Features |
|------|-------------|----------|
| Free | $0 | Basic scope definition, 2 projects, community support |
| Pro | $15 | Advanced scope management, 10 projects, priority support, collaboration tools, scope history |
| Enterprise | $50 | Unlimited projects, advanced analytics, custom integrations, dedicated support, API access |

## CAC Range
- Low-end: $120/customer (organic community growth, content marketing)
- High-end: $280/customer (paid advertising, partnerships)
- **Average CAC: $200/customer**

## LTV Estimate
- Average Revenue Per User: $25.50/month (70% Pro tier, 30% Enterprise tier)
- Gross Margin: 85%
- Monthly Churn Rate: 2.5%
- **LTV: $867** ($25.50 × 0.85 ÷ 0.025)

## Break-even Users Count
- Fixed Costs: $50,000/month (salaries, marketing, infrastructure)
- Contribution Margin per User: $20.50/month ($25.50 - $5.00)
- **Break-even: 2,439 active users**

## Path to $10K MRR
- Total Users Needed: 392
- Pro Tier Users (70%): 274 users × $15 = $4,110 MRR
- Enterprise Tier Users (30%): 118 users × $50 = $5,900 MRR
- **Total: $10,010 MRR**
- Timeline: 8-12 months (assuming 40-50 user additions/month)
